Detailed Description

template<typename EffectOperation>
class Effect< EffectOperation >

A template class that applies an effect operation to an Audio object.

This class takes an EffectOperation (a functor like Amplify, Normalize, FadeIn, FadeOut) and applies it to the samples of the base audio.

Template Parameters
EffectOperationThe type of effect to apply. It must be a callable object that takes a double (sample) and returns a double (modified sample), or for specific effects like FadeIn/FadeOut, it might take (sample_index, total_samples) and return a multiplier.
