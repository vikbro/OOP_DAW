Detailed Description

template<typename Generator>
class GeneratorAudio< Generator >

Template class for audio generated by a specific generator operation.

This class derives from Audio and uses a Generator functor (like SineGenerator) to produce audio samples on the fly. The audio data is not stored in a buffer but generated when requested via the operator[].

Template Parameters
GeneratorThe type of generator functor. This functor must implement sample operator()(std::size_t index) const to produce a sample at a given index.
