\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Generator$>$\newline
class Generator\+Audio$<$ Generator $>$}
Template class for audio generated by a specific generator operation. 

This class derives from {\ttfamily \mbox{\hyperlink{class_audio}{Audio}}} and uses a {\ttfamily Generator} functor (like {\ttfamily \mbox{\hyperlink{struct_sine_generator}{Sine\+Generator}}}) to produce audio samples on the fly. The audio data is not stored in a buffer but generated when requested via the {\ttfamily operator\mbox{[}\mbox{]}}.


\begin{DoxyTemplParams}{Template Parameters}
{\em Generator} & The type of generator functor. This functor must implement {\ttfamily sample operator()(std\+::size\+\_\+t index) const} to produce a sample at a given index. \\
