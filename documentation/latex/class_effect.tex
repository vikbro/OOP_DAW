\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{class_effect_operation}{Effect\+Operation}}$>$\newline
class Effect$<$ Effect\+Operation $>$}
A template class that applies an effect operation to an \mbox{\hyperlink{class_audio}{Audio}} object. 

This class acts as a decorator around an existing \mbox{\hyperlink{class_audio}{Audio}} object. It takes an {\ttfamily \mbox{\hyperlink{class_effect_operation}{Effect\+Operation}}} (a functor like \mbox{\hyperlink{struct_amplify}{Amplify}}, \mbox{\hyperlink{struct_normalize}{Normalize}}, \mbox{\hyperlink{struct_fade_in}{Fade\+In}}, \mbox{\hyperlink{struct_fade_out}{Fade\+Out}}) and applies it to the samples of the base audio.


\begin{DoxyTemplParams}{Template Parameters}
{\em \mbox{\hyperlink{class_effect_operation}{Effect\+Operation}}} & The type of effect to apply. It must be a callable object that takes a double (sample) and returns a double (modified sample), or for specific effects like Fade\+In/\+Fade\+Out, it might take (sample\+\_\+index, total\+\_\+samples) and return a multiplier. \\
\hline
\end{DoxyTemplParams}
