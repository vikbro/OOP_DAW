\doxysubsection{Detailed Description}
Represents an audio object whose data is primarily sourced from or destined for a file. 

This class extends the base {\ttfamily \mbox{\hyperlink{class_audio}{Audio}}} class to include a buffer for audio samples and methods for reading from and writing to various file formats (TXT, WAV).
