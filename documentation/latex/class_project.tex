\doxysubsection{Detailed Description}
Represents an audio project, which consists of multiple tracks. 

The \mbox{\hyperlink{class_project}{Project}} class manages a collection of {\ttfamily \mbox{\hyperlink{class_track}{Track}}} objects and provides functionalities for creating, saving, and loading projects.
