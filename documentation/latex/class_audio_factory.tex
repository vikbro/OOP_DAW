\doxysubsection{Detailed Description}
A singleton factory class for creating \mbox{\hyperlink{class_audio}{Audio}} objects. 

This class manages a collection of \mbox{\hyperlink{class_audio_creator}{Audio\+Creator}} objects and uses them to create instances of different \mbox{\hyperlink{class_audio}{Audio}} types.
